# Friction journal — "ugh, this again" log

One-week product-discovery experiment: press `F8` the moment something on
this PC feels like repeated friction (non-trading screen time especially),
type one line into the popup, get back to what you were doing. After a
week, review the list and run the demand/wedge/distribution filter to pick
the next build-for-self product.

Entry format (what F8 appends):

    - 2026-07-07 19:42 | [Google Chrome — Gmail] | copying the same trade numbers between three tabs again

Review with: `python tools/friction_review.py`
